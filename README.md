
# 🤖 Frontend Chatbot Application

A modern **frontend chatbot UI** built using **React + Vite**, designed with reusable components, clean structure, and scalable architecture.  
This project is ideal for learning **component-based frontend development** and showcasing a chatbot interface.

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository
```bash
git clone https://github.com/your-username/frontend-chatbot.git
cd frontend
````

### 2️⃣ Install Dependencies

```bash
npm install
```

### 3️⃣ Setup Environment Variables

Create a `.env` file in the root directory:

```env
VITE_API_URL=your_api_endpoint_here
```

### 4️⃣ Start Development Server

```bash
npm run dev
```

🔗 App runs on:

```
http://localhost:5173
```
